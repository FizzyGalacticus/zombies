Description:

	This is a zombie shooter game made for CS201 way back when...

Controls:

	Up Key - Jump
	Left/Right Keys - Move
	Up/Down Keys - Navigate Menus
	1-5 - Buy Items
	Z - Select/Shoot
	X - Repair Barricades
	Space - Pause
